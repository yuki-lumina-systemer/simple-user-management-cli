# simple-user-management-cli

A simple CLI user management system implemented in Java.

## Features
- Create user
- View user information (with PIN authentication)
- List users
- Change PIN (with authentication)

## How to Run
### Requirements
- Java 17+ (or your current Java version)

### Run
Compile and run `Main.java`.

Example (command line):
```bash
javac src/Main.java src/User.java src/UserService.java
java -cp src Main
